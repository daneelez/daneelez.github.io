Со словами год, раз употребляются порядковые числительные, поэтому меняем только последнюю цифру:

до тысяча восемьсот двенадцатого года.
к две тысячи восьмому году (но: в двухтысячном году).
с две тысячи первого раза.

Не путай с количественными числительными:

около тысячи станций, нет двух тысяч рублей, из двух тысяч солдат, нет тысячи причин.

Собирательные числительные

двое, трое, четверо, пятеро, шестеро, семеро

не употребляются с ж. р.,

НО:
– употребляются только с парными предметами:

двое брюк, пятеро ворот, двое влюблённых.

– с названиями детёнышей, с местоимениями:

двое волчат, двое из нас, двоими щенятами.

– имеют только мн. ч.:

трое суток, четверо ножниц, пятеро саней.
